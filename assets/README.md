# Assets Directory

This directory contains static assets for the FitCoach AI React Native app.

## Required Assets

To complete the app setup, you'll need to add the following assets:

### App Icons
- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (32x32)

### Splash Screen
- `splash.png` - Splash screen image (1242x2436 recommended)

## Asset Guidelines

- Use PNG format for icons and splash screens
- Ensure high resolution for crisp display on all devices
- Follow platform-specific guidelines for icon design
- Test on both iOS and Android devices

## Generating Assets

You can use tools like:
- [App Icon Generator](https://appicon.co/)
- [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)
- Adobe Illustrator/Photoshop for custom designs
